# Cognizant Digital Nurture 5.0 - JavaFSE

This repository contains the coursework and exercises completed during the Cognizant Digital Nurture 5.0 Java Full Stack Engineering (JavaFSE) program.

## Overview

The Cognizant Digital Nurture 5.0 JavaFSE program is designed to provide comprehensive training in Java full-stack development, covering fundamental concepts, frameworks, and best practices used in enterprise software development.

## Course Structure

### Week 1

**Algorithms and Data Structures**
- Implementation of fundamental algorithms
- Data structure operations and manipulations

**Design Patterns and Principles**
- Understanding and implementing design patterns
- SOLID principles and their applications
- Object-oriented design best practices

**JUnit Basic Testing Exercises**
- Unit testing fundamentals
- Test case design and implementation
- Assertion techniques

**Mockito Exercises**
- Mocking frameworks for testing
- Test doubles and stubs
- Integration testing with Mockito

**PLSQL Exercises**
- Database programming with PL/SQL
- Stored procedures and functions
- Database triggers and cursors

**SL4J Logging Exercises**
- Logging framework implementation
- Log level management
- Logging best practices

### Week 2

**1. Spring Core and Maven**
- Spring Framework fundamentals
- Dependency Injection and IoC Container
- Maven project configuration and management
- Library Management System implementation
- Spring Application Configuration

**2. Spring Data JPA**
- Spring Data JPA fundamentals
- Entity mapping and relationships
- Repository pattern implementation
- JPA vs Hibernate vs Spring Data JPA comparison
- Country management application with REST APIs

## Technologies Covered

- **Java**: Core language features and advanced concepts
- **Spring Framework**: Core, Data JPA
- **Maven**: Build automation and dependency management
- **JUnit**: Unit testing framework
- **Mockito**: Mocking framework for testing
- **PL/SQL**: Database programming
- **SL4J**: Logging framework
- **Design Patterns**: Gang of Four patterns
- **Data Structures**: Arrays, Lists, Maps, Sets, etc.
- **Algorithms**: Sorting, searching, optimization

## Getting Started

### Prerequisites
- Java Development Kit (JDK) 8 or higher
- Maven 3.6 or higher
- IDE (IntelliJ IDEA, Eclipse, or VS Code)
- Database (for PL/SQL and JPA exercises)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Yash-2808/Cognizant-Digital-Nurture-5.0-JavaFSE.git
```

2. Navigate to the project directory:
```bash
cd Cognizant-Digital-Nurture-5.0-JavaFSE
```

3. For Maven projects, navigate to the specific project directory and run:
```bash
mvn clean install
```
